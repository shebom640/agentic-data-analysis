# 🚀 Agentic Data Analysis

A sophisticated logistics analytics suite powered by a **100% deterministic Cascade Fuzzy Logic & Rule-Based Inference System**. 

## Overview
This platform bridges the gap between static operational metrics and dynamic resource allocation by substituting traditional black-box machine learning engines with an explainable, deterministic multi-agent pipeline. 

### Key Components
1. **Data Ingestion Engine**: Automatically parses CSV datasets, calculates system averages for operations, and normalizes values to auto-scale the simulation.
2. **Prediction Agent (Fuzzy Inference)**: Accepts `Time of Day` and `Customer Rating` to deterministically calculate and infer future `Service Demand`, resulting in a normalized `Order Flow` output.
3. **Decision Agent (Fuzzy Controller)**: Cascades the output of the Prediction Agent alongside `Kitchen Preparation Time (KPT)` and `Rider Waiting Time (RWT)`. It evaluates overlapping boundary memberships using a mathematical centroid mapping `(Σ μ × D) / Σ μ` to compute an optimal rider allocation strategy (`REDUCE`, `HOLD`, `SCALE UP`).

## Technology Stack
- **Frontend**: Flutter Web (Dart) – delivers a 60fps responsive dashboard with interactive sliders and continuous graphical charts.
- **Backend**: FastAPI (Python) – serves REST endpoints and implements the foundational fuzzy mathematical structures.
